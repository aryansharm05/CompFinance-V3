# CompFinance

CompFinance is a web application designed to help small and local businesses evaluate their financial performance through key financial ratios and industry benchmarks.

## Overview

The application allows businesses to:
1. Input their company information
2. Enter key financial data
3. Calculate important financial ratios
4. Compare their ratios against industry benchmarks

## Features

- **Company Information Input**: Collect basic company details including name, location, product type, and industry
- **Financial Data Input**: Input key financial metrics from income statements and balance sheets
- **Financial Ratio Calculations**: Automatically calculates important ratios including:
  - Profitability Ratios (Net Profit Margin, ROE, ROA)
  - Liquidity Ratios (Current Ratio, Quick Ratio)
  - Efficiency Ratios (Asset Turnover, Inventory Turnover)
  - Leverage Ratios (Debt to Equity)
- **Industry Benchmarking**: Compare company ratios against industry averages
- **Responsive Design**: Works on both desktop and mobile devices
- **Dark/Light Mode**: Toggle between dark and light themes

## Technical Details

The application is built using:
- HTML5
- CSS3
- Vanilla JavaScript

No external libraries or frameworks are used, making it lightweight and easy to deploy.

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. No build process or dependencies required

## Usage

1. Start on the Overview page
2. Click "Get Started" to begin
3. Enter company information
4. Input financial data
5. View calculated ratios
6. Compare with industry benchmarks
